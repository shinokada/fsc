## Getting started

```sh

```

## What in this?

Flowbite-Svelte has many components. This CLI allows you to find compoent default props from your command line.

## Commands

```sh
// list all Flowbite-Svelte components
fsc
// list export props of a component
fsc <component>
fsc badge
fsc accordiondefault
fsc button
// show help
fsc --help
// show version
fsc --version
```
